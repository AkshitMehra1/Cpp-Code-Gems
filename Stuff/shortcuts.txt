//Shortcuts


1. cltr+g : go to line

2. shift+alt+enter : fullscreen current code window.

3. F12 : Go to definition.

4. ctrl + å (] if english keyboard) : go to other side of parantheses.

5. ctrl + K, ctrl+D : auto format dokument.

6. ctrl + K, ctrl + F : auto format selection.

7. ctrl+shift+I : go backwards in the search rather than forward.

8. ctrl+m+o : collapse all sections.

9. ctrl+m+l : expand all sections.
